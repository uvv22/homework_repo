# Подсказки по GIT

## Создание репозитория
```sh
gti init
```
## Просмотр статуса
```sh
git status
```
## Индексация файла
```sh
git add
```
## Добавление комита
```sh
git commit -m ""
```
## Вывод сохраненных изменений
```sh
git log
```
### Вывод сохраненных изменений (усеченно)
```sh
git log --oneline
```

## Перемещение по веткам
```sh
git checkout
```

## Отображение всех веток
```sh
git branch
```

## Формирование новой ветки
```sh
git branch "имя ветки"
```

## Слияние веток
```sh
git merge имя ветки
```
